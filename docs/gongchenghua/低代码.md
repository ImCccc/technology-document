# 低代码

低代码平台优势:

1. 开发效率提升: 降低重复的劳动力;
2. 质量保障: 通过配置生成的页面, 无需反复的回归测试, 只需要验证核心功能;
3. 业务驱动: 通过给产品一个产品级的框架, 驱动业务发展;
